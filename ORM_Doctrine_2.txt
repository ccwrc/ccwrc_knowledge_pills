Podstawowe komendy konsolowe przy współpracy z Symfony LTS (php app/console zamiast php bin/console):

  $ php app/console doctrine:generate:entity - encję należy podlinkować w pliku z kontrolerem 
    poprzez use my_bundle\Entity\my_entity_class (zgadza się, 2x backslash)
  $ php app/console doctrine:generate:entities my_bundle - setery, getery, klasa repo, konstruktory relacji, itp.
  $ php app/console doctrine:schema:update --force -aktualizacja bazy po zmianach w encjach

----------------------------------------------------------------
Przydatne klasy:
  use Doctrine\Common\Collections\ArrayCollection;
    // umożliwia trzymanie relacji do wielu innych obiektów
    // http://www.doctrine-project.org/api/common/2.3/class-Doctrine.Common.Collections.ArrayCollection.html


----------------------------------------------------------------
Relacje pomiędzy encjami:
  http://doctrine-orm.readthedocs.io/projects/doctrine-orm/en/latest/reference/association-mapping.html

  Wiele do jednego - jednokierunkowa.
    // encja A wie o jednej przypisanej do siebie encji B, encja B nie wie nic o encjach A (może być przypisana do wielu).
       class A{
       /**
       * @ORM\ManyToOne(targetEntity="B")
       * @ORM\JoinColumn(name="B_id", referencedColumnName="id")
       */
       private $B;
       } // w encji B nie umieszcza się żadnej dodatkowej adnotacji
