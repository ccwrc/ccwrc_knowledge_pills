W akcji kontrolera można na dwa sposoby wskazać, jaki szablon ma być użyty:
- użyć metody render() należącej do bazowej klasy kontrolera
return $this->render('hello/greetings/index.html.twig', array('name' => $name));
- użyć adnotacji @Template().
/** @Template("SensioBlogBundle:Post:show.html.twig")
i w funkcji: return array('post' => $post);

ściezka widoku w TWIGu: {{ _self.getTemplateName() }}

{% extends "::base.html.twig" %} - dziedziczenie
{% block title %}Witaj w widoku {% endblock %} - zastapienie title
{{ parent() }} - treść z bloku nadrzędnego szablonu :)

{{ foo.bar }} - wyświetlanie atrybutu
{{ foo2['bar'] }} - Wyświetlenie zawartości komórki o kluczu bar z tablicy foo2
{{ var|escape|title }} - filtrów może być wiele
{% filter upper %} This text becomes uppercase {% endfilter %}

załączanie innego szablonu:
{% for post in posts%}
{{ include('post/show.html.twig',{ 'post': post})}}
{% endfor %}
